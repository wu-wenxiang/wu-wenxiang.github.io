## 缘起

昨天入手了Thinkpad T460s，因为只有USB3.0的缘故，U盘按照Windows各种失败，遂尝试Ubuntu。16.04已经可用，python默认是3.0，但，还是等等，14.04保险点。

## 安装步骤

### 安装基本系统
- 制作USB boot系统
- 检查`/etc/apt/sources`
- `sudo apt-get update`
- `sudo apt-get upgrade -y`

### 安装应用软件
- `sudo apt-get install git vim`
- `sudo apt-get install ibus-googlepinyin`
- chromium
- retext
